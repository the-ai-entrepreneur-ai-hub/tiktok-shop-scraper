# TikTok Shop Scraper

Find top-selling TikTok Shop affiliates and products by category. Discover trending products, commission rates, and top seller analytics for affiliate marketing research.

## Features

- Scrape TikTok Shop product listings by category
- Find top affiliate sellers and their commission rates
- Track trending products and bestsellers
- Extract product details, pricing, and sales data
- Filter by category, price range, and sales volume
- Monitor competitor affiliate strategies

- Build product sourcing lists for dropshipping
- Track TikTok Shop marketplace trends

## Input Example

```json
{
  "category": "Beauty & Personal Care",
  "sortBy": "bestSelling",
  "maxProducts": 100,
  "includeAffiliateData": true
}
```
